# Simple Python Calculator

Simple python calculator (for now)


List of mathematical constants you can use instead of numbers:
- `pi` (π) = 3.14159...
- `e` (Euler's number) = 2.71828...
- `tau` (τ = 2π) = 6.28318...
- `phi` (Golden ratio) = 1.61803...

## How to Use

1. Run the calculator: `python3 pycalc.py`
2. Choose whether to see the introduction (first run only)
3. Choose whether to show remainders for division
4. Enter an equation in the format: `number operator number`


### Supported Operators

- `+` Addition
- `-` Subtraction
- `*` Multiplication
- `/` Division (with optional remainder)
